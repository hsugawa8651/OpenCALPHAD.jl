Running Examples

# Navigate to the package directory
cd OpenCALPHAD.jl

# Run a single example
julia --project=. examples/101_gibbs_energy_agcu.jl

# Run interactively (keeps plot window open)
julia --project=. -i examples/411_phase_diagram_agcu.jl
